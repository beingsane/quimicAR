# quimicAR
Ensino de quimica utilizando realidade aumentada

# Instruções
#### Instalando depêndencias
 Executar ``` gradle clean build ``` na pasta raiz do projeto
#### Executando
 Executar ```gradle run``` na pasta do projeto
